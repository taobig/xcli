## Command line interface
```shell
./xcli install -h
sudo ./xcli install -r "-f,prod-config.yaml"  -r "--logLevel,debug"

sudo ./xcli uninstall
sudo ./xcli start
sudo ./xcli stop
sudo ./xcli restart

./xcli log
./xcli log -f
```
